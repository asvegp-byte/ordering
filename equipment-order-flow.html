What kind of job is this?

Sizing

Load & tonnage

Run the Manual J — don't blind-match the old unit's tonnage. Phoenix-tuned to ±10% so you don't oversize. Set the design airflow target (~400 CFM/ton) here; it drives blower and duct decisions later.

Route · Heating source

How is the space heated?

The heat source decides the indoor unit type — this is the first real fork.

Gas → Condenser + Furnace + Coil Electric → Condenser + Air Handler (heat kit) Heat pump → HP + Air Handler (+aux) Dual fuel → HP + Furnace + Coil
OUTPUT → Indoor unit type

Route · Configuration

Split or packaged — and orientation?

Split needs the indoor cabinet orientation locked; packaged needs the right package family and a curb/pad. Coil and blower orientation must match the airflow direction.

Split · upflow / downflow / horizontal L–R Packaged · gas-elec / HP / straight-cool Rooftop curb vs ground pad
OUTPUT → Cabinet config + orientation
4
Verify · Match-up

Refrigerant family + AHRI match

Never mix refrigerant families — R-410A legacy and A2L (R-32 / R-454B) components are not interchangeable. Confirm the condenser + coil/air handler are a listed AHRI-matched combination (required for rated efficiency and warranty). A2L: confirm the RDS is present/required and check line-set reuse rules.

OUTPUT → AHRI ref # + refrigerant family locked
HARD STOP
5
Gate · Electrical & phase

Confirm building power before ordering

Check the building's incoming service first, then match the equipment to it — not the other way around.

  • Service is single-phase 208–240V, 3-phase 208/230V, or 3-phase 460V — verified at the panel
  • Ordered unit's nameplate phase + voltage matches the building
  • MCA / MOCP read → disconnect, whip gauge, and breaker sized to match
OUTPUT → Confirmed V / phase + electrical BOM
HARD STOP
6
Gate · Physical access & fit

Will it physically get to where it lives?

Where does the indoor unit sit — attic, closet, garage, mech room, rooftop? Measure before you order, pulling dimensions from the install survey.

  • Attic: access opening W×H, narrowest point on the path, and every turn — vs each component's cabinet width (air handler / furnace / coil)
  • Closet / mech room: door width + service clearances
  • Condenser: pad space + setbacks; rooftop: curb match + crane/lift plan

If it doesn't fit: choose an alternate model / horizontal config, split into smaller components, or schedule enlarging the access — before the PO, not on install day.

OUTPUT → Fit confirmed or alternate selected
7
Verify · Airflow

Duct & airflow compatibility

Can the existing duct move the design CFM at acceptable static? Check return sizing, plenum/transition needs, and that the blower type (PSC vs ECM/variable) matches the staging you selected.

OUTPUT → Blower + duct mods confirmed
8
Verify · Controls

Thermostat & control wiring

Communicating vs conventional 24V. Confirm the existing wire count supports the system (e.g., the 5-wire run), and that the stat, inverter, and RDS are all compatible.

OUTPUT → Stat + wire plan
9
Build · Bill of materials

Accessory BOM — the things that get forgotten

Line set (size + length), disconnect, whip, pad / straps, condensate (pump, primary + secondary drain & float switch, secondary drain pan in attics per code), filter / rack, surge protector, thermostat, gas line + sizing, flue / vent category, plenum & transition sheet metal.

OUTPUT → Complete line-item BOM
✓
Gate · Pre-order release

Final check before the PO goes out

  • AHRI-matched combo
  • Refrigerant family consistent
  • Phase + voltage confirmed
  • Fits the access path
  • Cabinet orientation correct
  • Full accessory BOM listed
  • Cross-checked vs pricebook
  • All gates signed off

All boxes checked → release PO. Any gap → back to the gate that failed.
